A Universally Unique Identifier (UUID) is a 128-bit label used to uniquely identify objects in computer systems.

Currently only UUID v4 is available.

```js
import { uuidv4 } from 'std:uuid'

console.log(uuidv4()); // cff7a611-7964-434f-a0ab-2d19e81beb7a
```
